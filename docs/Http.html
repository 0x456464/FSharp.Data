( Refernece the DLL and open FSharp.Net:

1: #r "../bin/FSharp.Data.dll"

Sending simple request:

1: Http.Download("http://tomasp.net")

Specifying the GET method and get parameters:

1: Http.Download
2:   ( "http://www.htmlcodetutorial.com/cgi-bin/mycgi.pl", 
3:     query = ["test", "foo"], meth="GET")

Specifying query parameters and headers (using the default GET method):

1: Http.Download
2:   ( "http://api.themoviedb.org/3/search/movie",
3:     query   = [ "api_key", "6ce0ef5b176501f8c07c634dfa933cff"
4:                 "query", "batman" ],
5:     headers = [ "accept", "application/json" ])

Making POST request with some body:

1: Http.Download
2:   ( "http://www.htmlcodetutorial.com/cgi-bin/mycgi.pl", 
3:     meth="POST", body="test=foo")
